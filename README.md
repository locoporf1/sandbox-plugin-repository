# sandbox-plugin-repository
Download sandbox-intellij-plugin.jar file and using your IntelliJ IDE:<br>
1.- Go to Settings<br>
2.- Plugins<br>
3.- Gear icon<br>
4.- Select option "Intall from disk..."<br>
